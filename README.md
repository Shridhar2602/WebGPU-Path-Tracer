# WebGPU-Path-Tracer

## Features
- Primitives - Spheres, Quads, & Triangles
- Load .obj meshes
- BVH acceleration structure
- Materials - diffuse, mirror & dielectrics
- Compute shader + (Vertex + Fragment shader)
- Orbit Camera
- Stratified sampling to sample rays in a pixel
- Progressive Rendering
